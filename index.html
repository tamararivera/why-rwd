Why you should do responsive web design

If you are creating a website, given the amount of time and effort that it takes, I suppose that you want your site to reach as many people as possible.

One way to do this is making it responsive, so it can be enjoyed in many devices, even on those not planned like a TV or a refrigerator.

Making a responsive website means to create it in a way that will be able to show all of its information to the user on any browser and fluidly on any screen size. This will enable your website to adjust itself to any device and also prevents it to depend heavily on some new or poorly supported browser feature or some other variable that the user can disable like javascript, cookies or flash.

Also, being responsive helps the user enjoy the site in the way that he/she likes or wants, like enabling the resizing of the font or the zoom of the website.
